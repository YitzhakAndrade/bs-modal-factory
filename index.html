Bootstrap Modal Factory

This is a plugin to create boostrap modals dynamically.

Open modal

  $.modalFactory({
    title: 'Hello!', body: 'Put some content here.'
  }).modal('show');
Basic Example

Body content can be HTML.

Open modal

  $.modalFactory({
    id: 'myModalId',
    title: 'HTML Body Content'
    body: '<div class="alert alert-info">Some info</div>'
  }).modal('show');
Ajax Content

Open modal

  $.ajax({
    type: "GET",
    cache: false,
    url: "./ajax-content-example.html",
    success: function (data) {
      $.modalFactory({
        id: 'modalAjax',
        title: 'Ajax Content',
        body: data
      }).modal('show');
    }
  });
Options

Open modal

  $.modalFactory({
    id: 'myModalId',
    buttons: [
      {
        text: 'Click me...', cssClass: 'warning',
        onClick: function () { alert('Hello!') }
      },
      { text: 'Close modal', cssClass: 'danger', dataDismiss: true }
    ]
  }).modal('show');
